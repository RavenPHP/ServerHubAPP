# ServerHubAPP

*Creator:* DevRaven  
*Version:* 1.0

ServerHub is a multiplatform Kotlin app for sharing and discovering Discord servers. Users can add servers, search communities, and see only verified bots.

## Features

- Share Discord servers with invite links
- Search servers by name or description
- Categories: Gaming, Education, etc.
- Verified bot indicator ✅
- Works on Android & iOS
- Simple, clean, and multiplatform UI

## Planned Features

- User accounts and login
- Admin panel for verification
- Trending servers
- Comments & likes
- Notifications for new servers

## Technology

- Kotlin Multiplatform
- Compose Multiplatform for UI
- Shared logic for Android and iOS
- Simulated backend repository
